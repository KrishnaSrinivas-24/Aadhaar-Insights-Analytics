► Temporal Patterns:

6.2 ANOMALY DETECTION MODEL
---------------------------

[See Chart: anomaly_detection_results.png]

PURPOSE:
Identify unusual patterns in Aadhaar data that may indicate system errors,
data quality issues, or potential fraudulent activity.

METHODOLOGY:
- Algorithm: Isolation Forest (unsupervised ML)
- Features: Total count, Z-score, State deviation, District deviation

- Destination states: Maharashtra, Gujarat, Tamil Nadu (in-migration)

INSIGHT 3: SEASONAL DEMAND PATTERNS
-----------------------------------
Clear seasonal patterns enable predictive planning:
- Q1 (Jan-Mar): HIGH demand for demographic updates (tax compliance)
- Q4 (Oct-Dec): Rising activity before new year
- July: Peak for enrolment and biometric (special drives)

INSIGHT 4: INFRASTRUCTURE QUALITY SIGNALS
-----------------------------------------
Maharashtra's disproportionate biometric update rate suggests potential issues:
- Equipment quality review needed
- Operator retraining recommended
- Process standardization required

INSIGHT 5: FAMILY MIGRATION PATTERNS

SAMPLE CODE: Data Loading and Preprocessing
-------------------------------------------

```python
import pandas as pd
import glob

# Load all enrolment CSV files
files = glob.glob('datasets/api_data_aadhar_enrolment/*.csv')
df_list = [pd.read_csv(f) for f in files]
df = pd.concat(df_list, ignore_index=True)

# Data cleaning
df.drop_duplicates(inplace=True)
df.fillna(0, inplace=True)

# Feature engineering
df['date'] = pd.to_datetime(df['date'], dayfirst=True)
df['total'] = df['age_0_5'] + df['age_5_17'] + df['age_18_greater']
```

SAMPLE CODE: Anomaly Detection
------------------------------

```python
from sklearn.ensemble import IsolationForest

# Create features
features = ['total', 'z_score', 'state_deviation']
X = df[features].fillna(0)

# Train Isolation Forest
model = IsolationForest(contamination=0.01, random_state=42)
predictions = model.fit_predict(X)

# Flag anomalies
df['is_anomaly'] = (predictions == -1).astype(int)
```
